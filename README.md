# rust_with_azure
MS Azure LOVE [not] Rust

Azure + Rust: High-Performance Procurement Automation
This project explores the synergy between Microsoft Azure and Rust — building robust, high-throughput services that interface with MSSQL databases and execute complex business logic for automated procurement workflows.

🚀 What's Inside
Azure-native services built with Rust

MSSQL database integration — efficient, type-safe, and reliable

High-throughput data processing — designed to handle large-scale, real-world procurement scenarios

Complex business logic — automating purchasing decisions, approvals, and workflows

🎯 Purpose
This repository serves as a playground and proving ground for combining Azure's cloud ecosystem with Rust's performance and safety guarantees. The focus is on building production-ready components that can scale under heavy load while maintaining correctness and maintainability.

🛠️ Tech Stack
Language: Rust 🦀

Cloud: Microsoft Azure (Functions, Event Hubs, etc.)

Database: MSSQL (with proper async/connection pooling)

Domain: Automated procurement systems
